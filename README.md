 Meta Ads Performance Analysis

 Project Overview
This project analyzes Meta (Facebook & Instagram) ad performance to evaluate campaign effectiveness and optimize marketing strategy.

 Tools Used
- Power BI
- Excel / CSV

 Key Metrics
- Impressions
- Clicks
- CTR (Click Through Rate)
- CPC (Cost Per Click)
- Conversions

 Dashboard Features
- Campaign performance analysis
- CTR and CPC comparison
- Platform-wise analysis (Facebook vs Instagram)
- Trend analysis over time

 Insights
- Identified high-performing campaigns
- Improved understanding of ad efficiency
- Found areas to reduce cost and increase conversions
